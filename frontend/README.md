# ByGC — Finanças

App de controle financeiro pessoal interativo.

**Desenvolvido por Gabriel e Caio**

## Como rodar

```bash
npm install
npm run dev
```

Abra [http://localhost:5173](http://localhost:5173) no navegador.

## Funcionalidades

- Dashboard com saldo, receitas e despesas
- Gráfico de pizza por categoria de despesas
- Gráfico de barras com histórico mensal
- Adicionar, editar e excluir transações
- Filtros e busca de transações
- Dados salvos automaticamente no navegador (localStorage)
